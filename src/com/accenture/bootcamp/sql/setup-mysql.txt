1. Install docker:
    https://www.docker.com/products/docker-desktop
    Windows:    https://docs.docker.com/docker-for-windows/install/
    MacOS:      https://docs.docker.com/docker-for-mac/install/

2. Run this command in terminal:
    docker run --name bootcamp-mysql -p 3306:3306 -d -e MYSQL_ALLOW_EMPTY_PASSWORD=true mysql:8.0.13 --default-authentication-plugin=mysql_native_password

3. You can connect to your database in two ways:
   * Through Intellij IDEA:
          https://www.jetbrains.com/help/idea/connecting-to-a-database.html
          https://www.youtube.com/watch?v=AshoqjeaPkc
   * Or by using MySql workbench - https://dev.mysql.com/downloads/workbench/
